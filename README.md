Программа принимает на вход параметры -t, -c, -e. Параметр -t отвечает за тип связного списка: 1 это - однонаправленный связный список, 2 это – двунаправленный связный список, 3 это – однонаправленный кольцевой связный список, 4 это – двунаправленный кольцевой связный список. Параметр -c отвечает за количество элементов из которых должен быть сформирован связный список, а параметр -e описывать сами элементы связного списка через запятую.	Программа содержит меню выбора действия. За символ завершения программы принимается 0.Параметры должны указываться через запятую без пробелов например: "-t 1 -c 0 -e".
В файле необходимо на первой строке указать особый символ: ">" - однонаправленный связанный список "<>" - двунаправленный связный список ">0" - однонаправленный кольцевой связный список "<>0" - двунаправленный кольцевой связный список А на второй строке через пробел записать элементы списка
